Testing About html

HTML stands for Hypertext markup language. It refers the way in which webpages that is HTML documents are linked together thus the link available on a webpage is called HyperText. There are two types of tags available in HTML:

1. Empty tags.
2. Container tags.

Some of empty tags are:-

1. br Tag.
2. hr Tag.

Some of container tags are:-

1. HTML Tag.
2. BODY Tag.
